# News_Aggregator
A web app automatically collects news from NYTimes, Swiss Info, and Ettoday. Created by Flask.

## Environment
- python 3.8
- Flask
- bs4
- request
- request_html

## Setup
Directly run

```python3 app.py```

The app is running on http://0.0.0.0:5000/ by default.
